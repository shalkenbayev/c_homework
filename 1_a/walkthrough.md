# Обзор реализации — Кусочно-заданная функция

Я успешно реализовал калькулятор кусочно-заданной функции на языке C, включая блок-схему и обработку ошибок.

## Компоненты

### 1. Алгоритм и блок-схема
Логика использует последовательные операторы `if` для обработки различных ветвей функции:
- $x < 4 \implies f(x) = \cos(x)$
- $x = 4 \implies f(x) = x^3 - bx + 1$
- $4 < x < 7 \implies f(x) = \sqrt{ax}$ (с проверкой на $ax \ge 0$)
- $x \ge 7 \implies$ Ошибка (нарушение области определения)

### 2. Программа на C
Исходный код находится в файле [main.c](main.c). Он содержит комментарии для ясности (около 20% строк кода).

## Инструкция по запуску

Для запуска программы на Mac выполните следующие действия:

1. **Компиляция**:
   ```bash
   gcc -o piecewise main.c -lm
   ```
2. **Запуск**:
   ```bash
   ./piecewise
   ```

## Результаты тестирования

Программа была протестирована на пяти различных наборах данных для проверки всех ветвей логики.

| Ввод (a, b, x) | Ожидаемый результат / Ошибка | Фактический результат | Статус |
| :--- | :--- | :--- | :--- |
| `0 0 3.0` | $f(3.0) = \cos(3.0) \approx -0.99$ | `Значение f(x) равно: -0.9900` | ✅ Ок |
| `0 5 4.0` | $f(4.0) = 4^3 - 5(4) + 1 = 45$ | `Значение f(x) равно: 45.0000` | ✅ Ок |
| `2 0 5.0` | $f(5.0) = \sqrt{2 \cdot 5} \approx 3.1623$ | `Значение f(x) равно: 3.1623` | ✅ Ок |
| `-2 0 5.0` | Ошибка: $ax < 0$ | `Ошибка: Выражение под корнем отрицательно.` | ✅ Ок |
| `0 0 8.0` | Ошибка: $x \ge 7$ | `Ошибка: Функция определена только для x < 7.` | ✅ Ок |
