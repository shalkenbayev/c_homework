# Обзор решения — Обработка матриц (Вариант 16)

Я реализовал модульную программу на языке C для обработки матриц $n \times n$ в соответствии с логикой Варианта 16.

## Как запустить и протестировать

### 1. Компиляция
Используйте компилятор `gcc` для сборки программы:
```bash
gcc -Wall -Wextra matrix_task.c -o matrix_task
```

### 2. Запуск
Запустите исполняемый файл:
```bash
./matrix_task
```

### 3. Тестирование (Рекомендуемые наборы данных)

#### Тест 1: Четная матрица ($n=4$)
- Введите размер: `4`
- Выберите метод заполнения: `2` (Случайно)
- Введите порог: `0`
- **Что проверить**: Убедитесь, что элементы в Секторе 5 (верх и низ) выделены `*`, а после обработки элементы Сектора 3\ (низ и диагональ) стали `0`.

#### Тест 2: Нечетная матрица ($n=5$)
- Введите размер: `5`
- Выберите метод заполнения: `1` (Вручную)
- Введите элементы (можно небольшие числа)
- **Что проверить**: Проверьте корректность суммы в Секторе 5 и правильность обнуления диагонали и нижнего треугольника.

```text
Исходная матрица:
   11  * -44* * -31*   -28  
   47     21     -2     15

Часть 1 (Сектор 5): Сумма элементов > 10 = 47
```

## Пояснение логики секторов
- **Сектор 5**: Область выше главной и побочной диагоналей (Верх) И область ниже обеих диагоналей (Ниж).
- **Сектор 3\**: Область ниже обеих диагоналей (Ниж) ВКЛЮЧАЯ главную диагональ.
