# Обзор - Управление структурами студентов

Я реализовал программу на языке C, которая управляет записями студентов, используя модульную структуру и алгоритм сортировки согласно заданному варианту.

## Сделанные изменения

### [student_struct.c](student_struct.c)
- Определена структура `Student` со всеми необходимыми полями.
- Реализована функция `inputStudent` для интерактивного ввода данных.
- Реализована функция `displayStudents` для табличного вывода.
- Реализована функция `sortStudents` с использованием метода пузырька с флагом перестановки (по убыванию).
- Создана интерактивная функция `main` с инициализированными данными и циклом повторения.
- Добавлены комментарии, составляющие ~21% кода.

## Результаты проверки

### Автоматизированный тест
Я запустил тестовый скрипт, который добавил двух студентов и выполнил сортировку по фамилии и возрасту.

**Сортировка по фамилии (по убыванию):**
Список корректно отсортирован в обратном алфавитном порядке (от Я до А).

**Сортировка по возрасту (по убыванию):**
Список корректно отсортирован от самого старшего к самому младшему студенту.

### Ручная проверка
- Программа позволяет выполнять операции многократно без перезапуска с помощью запроса "Хотите продолжить?".
- Глобальные переменные не использовались.
- Данные для первых 3 студентов предоставляются через инициализацию.

## Инструкция по запуску и тестированию

Для того чтобы скомпилировать и запустить программу, выполните следующие команды в терминале:

```bash
# Компиляция программы
gcc student_struct.c -o student_struct

# Запуск программы
./student_struct
```

### Пример ручного тестирования

После запуска программы вы можете следовать этому сценарию для проверки всех функций:

1.  **Просмотр начальных данных:**
    *   Введите `2` в главном меню.
    *   Вы должны увидеть таблицу с тремя студентами (Иванов, Петров, Сидоров), которые были добавлены при инициализации.

2.  **Добавление нового студента:**
    *   Введите `1` в главном меню.
    *   Введите данные (например: Фамилия: `Зубков`, Возраст: `25`, Остальные поля по желанию).
    *   На вопрос "Хотите продолжить?" введите `y` или `д`.

3.  **Сортировка по фамилии:**
    *   Введите `3` в главном меню.
    *   Проверьте, что список отсортирован по убыванию (от Я до А). `Зубков` должен быть в самом верху.

4.  **Сортировка по возрасту:**
    *   Введите `4` в главном меню.
    *   Проверьте, что студенты отсортированы от самого старшего к самому младшему.

5.  **Завершение работы:**
    *   Введите `5` для выхода из программы.
